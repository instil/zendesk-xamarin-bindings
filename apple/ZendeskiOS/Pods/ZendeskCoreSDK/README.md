# Zendesk Core SDK for iOS

Zendesk Core SDK for mobile. The Core SDK is a shared library among the Zendesk product SDKs. It is used to shared core functionality between our products like identity and storage.

## Documentation

You can find our documentation on https://developer.zendesk.com/embeddables.
